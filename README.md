# tetris-pygame
Um jogo de Tetris desenvolvido em Python, utilizando a biblioteca Pygame. Possui um sistema de peças com rotação, detecção de colisões, pontuação dinâmica e aumento de níveis progressivo, efeitos sonoros e uma interface de game over que mostra a pontuação final do jogador. Desenvolvido como parte do meu portfólio e estudo de lógica de jogos.
